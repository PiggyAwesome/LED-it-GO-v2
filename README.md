# LED-it-GO-v2
Simple Python program to leak data using coloured screen flashes

### This program does:
+ Open a txt file
+ Read the contents
+ Convert the contents to binary
+ Output the binary using coloured image flashes

### Flashes meaning:

> Black = 0 
>
> White = 1
>
> Blue = "Switching to next value"
>
